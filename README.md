# 🏃‍♂️ AME - Agencia de Movimiento y Energía

Página web educativa para el proyecto de Educación Física con temática de agentes secretos.

## 📁 Estructura del Proyecto

```
Agencia-AME/
├── index.html          # Página principal
├── logo-ame.png        # Logo de la agencia
├── escalera.png        # Imagen decorativa
└── data/               # Archivos PDF descargables
    ├── EV1/            # PDFs de 1ª Evaluación (UD 1-4)
    ├── EV2/            # PDFs de 2ª Evaluación (UD 5-8)
    └── EV3/            # PDFs de 3ª Evaluación (UD 9-12)
```

## 📚 Unidades Didácticas

### 1ª Evaluación (UD 1-4)
- **UD 1**: Protocolo Agente Saludable
- **UD 2**: Activación de las Capacidades AME
- **UD 3**: Entrenamiento Conjunto - Cooperación en Acción
- **UD 4**: Archivo Escalones Invisibles

### 2ª Evaluación (UD 5-8)
- **UD 5**: Activación de las Capacidades II
- **UD 6**: Agente en Terreno - Orientación Estratégica
- **UD 7**: Entrenamiento de Campo - Floorball
- **UD 8**: Patrones Codificados AME - Tutting

### 3ª Evaluación (UD 9-12)
- **UD 9**: Operación Tradición - Juegos Tradicionales
- **UD 10**: Entrenamiento de Campo - Cachibol
- **UD 11**: Coordinación Acrobática AME
- **UD 12**: Agente en Terreno - Voleibol

## 🚀 Cómo Añadir PDFs

1. Coloca los PDFs en la carpeta correspondiente según la evaluación:
   - `data/EV1/` para UDs 1-4
   - `data/EV2/` para UDs 5-8
   - `data/EV3/` para UDs 9-12

2. Los archivos se mostrarán automáticamente en la web

3. Los alumnos podrán descargarlos desde cada unidad didáctica

## 🎨 Características

- ✅ Diseño temático de agencia secreta
- ✅ Navegación por evaluaciones y unidades
- ✅ Descarga directa de materiales PDF
- ✅ Responsive (funciona en móviles)
- ✅ Sin necesidad de sistema de login
- ✅ Página estática (fácil de mantener)
